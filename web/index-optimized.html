How It Works

1. Input Vector

Start with a continuous vector in ℝⁿ space. This represents the agent's position or state.

input.rs 
let vec = [0.6f32, 0.8];

2. Φ-Folding (Pythagorean Manifold)

The continuous vector is folded onto a pre-computed manifold of exact Pythagorean geometric states.

manifold.rs 
// Create manifold with 200 Pythagorean triples
let manifold = PythagoreanManifold::new(200);

// Pre-build KD-tree index for O(log n) queries
manifold.build_index();

3. Snapping

The vector is snapped to the nearest exact geometric state. Guaranteed noise < 0.001.

snap.rs 
// Snap continuous vector to exact discrete state
let (snapped, noise) = snap(&manifold, vec);

assert!(noise < 0.001);  // Guaranteed exact result
println!("Snapped: ({}, {})", snapped[0], snapped[1]);
// Output: (0.6, 0.8) = (3/5, 4/5) exactly

4. Spatial Query

Query the KD-tree to find all agents within the geometric radius. O(log n) time complexity.

query.rs 
// Query neighbors within radius
let radius = 0.2;
let neighbors = manifold.neighbors(snapped, radius);

// Returns: O(log n) time complexity
// Each agent sees only its local neighborhood
println!("Found {} neighbors", neighbors.len());
